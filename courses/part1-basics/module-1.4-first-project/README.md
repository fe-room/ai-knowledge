# 模块1.4：第一个AI项目

> 完成你的第一个AI应用程序，体验从0到1的成就感

## 学习目标

完成本模块后，你将能够：
- 使用Python和JavaScript调用AI API
- 构建一个简单的聊天机器人
- 理解AI应用的基本结构
- 独立完成一个小型AI项目

**课时安排**：2课时

## 项目介绍

### 项目目标

构建一个命令行聊天机器人，具有以下功能：
- 多轮对话能力
- 流式输出（打字机效果）
- 退出命令

### 技术要点

- Python：openai库、异步编程
- JavaScript：openai库、async/await
- 环境变量管理

## 学习建议

**初学者**：
- 先完成一个语言版本（推荐Python）
- 理解每一行代码的作用
- 尝试修改参数观察效果

**有基础学员**：
- 两个版本都完成
- 对比Python和JavaScript的实现差异
- 尝试添加新功能

## 完成标准

- ✅ 项目能够成功运行
- ✅ 能够进行多轮对话
- ✅ 能够正确退出程序
- ✅ 理解代码的基本结构
