## 一、API调用流程

### 1.1 基本流程图

```
┌──────────────┐
│   准备密钥    │
│  API Key     │
└──────┬───────┘
       ↓
┌──────────────┐
│   创建客户端   │
│   Client     │
└──────┬───────┘
       ↓
┌──────────────┐
│   构建请求    │
│  Request     │
└──────┬───────┘
       ↓
┌──────────────┐
│   发送请求    │
│   API Call   │
└──────┬───────┘
       ↓
┌──────────────┐
│   处理响应    │
│  Response    │
└──────────────┘
```

### 1.2 请求结构

**基本请求要素**：
```json
{
  "model": "模型名称",
  "messages": [
    {"role": "user", "content": "用户输入"}
  ],
  "temperature": 0.7,
  "max_tokens": 1000
}
```

**JavaScript**：

### 2.3 基础调用示例

**Python示例**：

```python
from openai import OpenAI
from dotenv import load_dotenv
import os

# 加载环境变量
load_dotenv()

# 创建客户端
client = OpenAI(
    api_key=os.getenv("OPENAI_API_KEY")
)

# 发送请求
response = client.chat.completions.create(
    model="gpt-3.5-turbo",
    messages=[
        {"role": "system", "content": "你是一个友好的助手"},
        {"role": "user", "content": "你好！"}
    ]
)

# 获取回复
reply = response.choices[0].message.content
print(reply)
```

**JavaScript示例**：

```javascript
import OpenAI from 'openai';
import dotenv from 'dotenv';

dotenv.config();

// 创建客户端
const client = new OpenAI({
  apiKey: process.env.OPENAI_API_KEY
});

// 发送请求
async function chat() {
  const response = await client.chat.completions.create({
    model: 'gpt-3.5-turbo',
    messages: [
      { role: 'system', content: '你是一个友好的助手' },
      { role: 'user', content: '你好！' }
    ]
  });

  console.log(response.choices[0].message.content);
}

chat();
```

### 2.4 重要参数说明

**model（模型选择）**：
- `gpt-3.5-turbo`：快速、便宜
- `gpt-4`：更强但更贵
- `gpt-4-turbo`：GPT-4的性能，更长上下文

**messages（消息列表）**：
- `system`：设置AI角色和行为
- `user`：用户输入
- `assistant`：AI之前的回复（用于多轮对话）

**temperature（随机性）**：
- 范围：0-2
- 0：确定性输出
- 0.7：平衡（推荐）
- 1+：更有创意但不可预测

**max_tokens（最大输出长度）**：
- 限制输出Token数量
- 影响成本

### 2.5 多轮对话

```python
messages = [
    {"role": "system", "content": "你是一个友好的助手"}
]

while True:
    user_input = input("你: ")
    if user_input.lower() in ["退出", "exit", "quit"]:
        break

    # 添加用户消息
    messages.append({"role": "user", "content": user_input})

    # 调用API
    response = client.chat.completions.create(
        model="gpt-3.5-turbo",
        messages=messages
    )

    # 获取回复
    reply = response.choices[0].message.content
    print(f"AI: {reply}")

    # 添加AI回复到历史
    messages.append({"role": "assistant", "content": reply})
```

### 2.6 流式响应

```python
# 流式输出
stream = client.chat.completions.create(
    model="gpt-3.5-turbo",
    messages=[{"role": "user", "content": "写一首诗"}],
    stream=True  # 开启流式
)

for chunk in stream:
    if chunk.choices[0].delta.content:
        print(chunk.choices[0].delta.content, end="", flush=True)
```


**JavaScript**：

```bash
npm install @anthropic-ai/sdk
```

### 3.3 基础调用示例

**Python示例**：

```python
from anthropic import Anthropic
from dotenv import load_dotenv
import os

load_dotenv()

# 创建客户端
client = Anthropic(
    api_key=os.getenv("ANTHROPIC_API_KEY")
)

# 发送请求
message = client.messages.create(
    model="claude-3-5-sonnet-20241022",
    max_tokens=1000,
    messages=[
        {"role": "user", "content": "你好！"}
    ]
)

print(message.content[0].text)
```

### 3.4 Claude与OpenAI的差异

| 特性 | OpenAI | Claude |
|------|--------|--------|
| System Prompt | messages中的system | system参数 |
| 模型名称 | gpt-3.5-turbo | claude-3-5-sonnet |
| 输出获取 | choices[0].message.content | content[0].text |
| 最大Token | 默认回答完整 | 必须指定max_tokens |

### 3.5 Claude的System Prompt

```python
message = client.messages.create(
    model="claude-3-5-sonnet-20241022",
    max_tokens=1000,
    system="你是一个专业的Python编程助手，擅长解释代码和调试问题。",  # System Prompt
    messages=[
        {"role": "user", "content": "如何读取JSON文件？"}
    ]
)
```

## 四、API计费理解

### 4.1 Token计费模式

**计费方式**：按Token数量计费

**价格示例（2024年）**：

| 模型 | 输入价格 | 输出价格 |
|------|---------|---------|
| GPT-3.5-Turbo | $0.50/1M tokens | $1.50/1M tokens |
| GPT-4 | $30/1M tokens | $60/1M tokens |
| Claude 3 Haiku | $0.25/1M tokens | $1.25/1M tokens |
| Claude 3.5 Sonnet | $3/1M tokens | $15/1M tokens |

### 4.2 成本计算示例

```python
# 示例：计算一次调用的成本

# 输入：1000 tokens
# 输出：500 tokens
# 模型：GPT-3.5-Turbo

输入成本 = 1000 / 1,000,000 * $0.50 = $0.0005
输出成本 = 500 / 1,000,000 * $1.50 = $0.00075
总成本 = $0.00125

# 大约0.1美分
```

### 4.3 成本优化建议

1. **选择合适的模型**：
   - 简单任务用GPT-3.5/Claude Haiku
   - 复杂任务用GPT-4/Claude Sonnet

2. **优化提示词**：
   - 精简不必要的文字
   - 使用模板复用

3. **控制输出长度**：
   - 设置合理的max_tokens
   - 要求简洁回答

4. **使用缓存**：
   - 相同请求使用缓存结果
   - 减少重复调用

## 五、错误处理

### 5.1 常见错误类型

**认证错误（401）**：
```python
# API密钥无效或过期
openai.AuthenticationError: Incorrect API key provided
```

**速率限制（429）**：
```python
# 请求过于频繁
openai.RateLimitError: Rate limit exceeded
```

**服务错误（500/503）**：
```python
# 服务器问题
openai.APIError: The server had an error
```

**内容过滤**：
```python
# 内容违反政策
openai.BadRequestError: Content was filtered
```

### 5.2 错误处理最佳实践

```python
import time
from openai import OpenAI, APIError, RateLimitError, APIConnectionError

client = OpenAI()

def call_api_with_retry(messages, max_retries=3):
    for attempt in range(max_retries):
        try:
            response = client.chat.completions.create(
                model="gpt-3.5-turbo",
                messages=messages
            )
            return response

        except RateLimitError as e:
            if attempt < max_retries - 1:
                wait_time = 2 ** attempt  # 指数退避
                print(f"速率限制，等待{wait_time}秒后重试...")
                time.sleep(wait_time)
            else:
                raise Exception("达到最大重试次数")

        except APIConnectionError as e:
            print(f"网络错误: {e}")
            if attempt < max_retries - 1:
                time.sleep(2)
            else:
                raise

        except APIError as e:
            print(f"API错误: {e}")
            raise
```

### 5.3 超时设置

```python
# 设置请求超时

    timeout=30.0,  # 30秒超时
    max_retries=2   # 最大重试次数
```

## 六、最佳实践

### 6.1 安全实践

1. ✅ 使用环境变量存储密钥
2. ✅ 定期轮换密钥
3. ✅ 使用最小权限密钥
4. ❌ 不要在日志中打印密钥
5. ❌ 不要在前端代码中暴露密钥

### 6.2 性能实践

1. **异步调用**：使用async版本提高并发
2. **流式响应**：大输出使用流式
3. **批量处理**：合并多个请求
4. **缓存结果**：避免重复调用

### 6.3 代码组织建议

```python
# utils/llm_client.py
from openai import OpenAI
from anthropic import Anthropic
import os

class LLMClient:
    def __init__(self):
        self.openai_client = OpenAI(api_key=os.getenv("OPENAI_API_KEY"))
        self.claude_client = Anthropic(api_key=os.getenv("ANTHROPIC_API_KEY"))

    def chat_openai(self, messages, model="gpt-3.5-turbo", **kwargs):
        """OpenAI聊天"""
        return self.openai_client.chat.completions.create(
            model=model,
            messages=messages,
            **kwargs
        )

    def chat_claude(self, messages, model="claude-3-5-sonnet-20241022", **kwargs):
        """Claude聊天"""
        return self.claude_client.messages.create(
            model=model,
            messages=messages,
            **kwargs
        )
```
