# 第一部分：基础篇

> 从零开始构建AI应用开发的知识基础

## 模块概览

本部分共4个模块，约20-30课时，帮助学员建立AI应用开发的完整知识基础。

**学习目标**：
- 掌握Python和JavaScript编程基础
- 理解AI、大语言模型和Agent的核心概念
- 配置开发环境并学会调用AI API
- 完成第一个AI应用程序

**适用人群**：
- 零基础学员：从编程基础开始，完整学习所有模块
- 有编程基础学员：重点学习AI概念和Agent原理（模块1.2），快速浏览其他模块

## 学习建议

**零基础学员**：
1. 按顺序完整学习所有模块
2. 每个模块的练习题都要完成
3. 重点理解异步编程概念（为后续Agent开发做准备）
4. 完成入门项目，建立信心

**有编程基础学员**：
1. 快速浏览模块1.1，重点看异步编程部分
2. 深入学习模块1.2（AI概念）
3. 学习模块1.3的API调用和密钥管理
4. 完成入门项目，熟悉AI API调用流程

## 学习资源

每个模块包含：
- 📖 **教程文档**：详细的理论讲解和概念说明
- 💻 **代码示例**：Python和JavaScript双语言实现
- ✍️ **练习题**：巩固知识点的练习
- 🎯 **实战项目**：模块1.4的入门项目
