### 题目7：类定义

创建一个 `BankAccount` 类：
- 属性：账户名、余额
- 方法：存款、取款、查询余额
- 取款时余额不能为负

使用示例：
```python
account = BankAccount("Alice", 1000)
account.deposit(500)
account.withdraw(200)
print(account.get_balance())  # 应该是1300
```

### 题目8：装饰器

使用示例：

### 题目10：并发控制

编写一个批量处理函数：
- 可以并发处理多个任务
- 支持设置并发数量限制
- 处理完成后返回所有结果

使用示例：
```python
async def process_item(item):
    await asyncio.sleep(0.5)
    return f"处理完成: {item}"

results = await batch_process(items, process_item, batch_size=3)
```
