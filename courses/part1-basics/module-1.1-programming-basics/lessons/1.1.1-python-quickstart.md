### 什么是Python？

Python是一门简洁、易读、功能强大的编程语言，由Guido van Rossum于1991年创建。

**Python的特点**：
- ✅ 语法简洁，接近自然语言
- ✅ 丰富的第三方库生态
- ✅ 跨平台支持
- ✅ 强大的社区支持

**Python在AI领域的应用**：
- 机器学习与深度学习（TensorFlow、PyTorch、scikit-learn）
- 自然语言处理（NLTK、spaCy、transformers）
- 数据分析与可视化（Pandas、NumPy、Matplotlib）
- AI应用开发（LangChain、OpenAI SDK、Anthropic SDK）
- Web开发（FastAPI、Flask）

## 三、基本语法

Python是动态类型语言，不需要声明变量类型。


### 3.2 运算符

```python
# 数学运算
a = 10
b = 3

print(a + b)   # 加法：13
print(a - b)   # 减法：7
print(a * b)   # 乘法：30
```

## 四、数据结构

### 4.1 列表（List）

列表是可变的有序集合。

```python
# 创建列表
fruits = ["apple", "banana", "cherry"]
numbers = [1, 2, 3, 4, 5]
mixed = [1, "hello", 3.14, True]

# 列表操作
fruits.append("orange")      # 添加元素
fruits.insert(1, "grape")    # 在索引1插入
fruits.remove("banana")      # 删除元素
last = fruits.pop()          # 弹出最后一个元素

# 列表切片
numbers = [0, 1, 2, 3, 4, 5]
print(numbers[1:4])    # [1, 2, 3]
print(numbers[:3])     # [0, 1, 2]
print(numbers[-2:])    # [4, 5]

# 列表长度
print(len(fruits))     # 4
```

### 4.2 字典（Dictionary）

字典是键值对的无序集合。

```python
# 创建字典
person = {
    "name": "Alice",
    "age": 25,
    "city": "Beijing"
}

# 访问元素
print(person["name"])          # "Alice"
print(person.get("age"))       # 25
print(person.get("job", "N/A")) # "N/A"（不存在则返回默认值）

# 修改元素
person["age"] = 26
person["job"] = "Developer"

# 删除元素
del person["city"]

# 遍历字典
for key, value in person.items():
    print(f"{key}: {value}")
```

### 4.3 集合（Set）

集合是无序的不重复元素集合。

```python
# 创建集合
colors = {"red", "green", "blue"}
numbers = {1, 2, 3, 3, 3}  # 自动去重：{1, 2, 3}

# 集合操作
colors.add("yellow")
colors.remove("green")

# 集合运算
a = {1, 2, 3}
b = {2, 3, 4}
print(a | b)   # 并集：{1, 2, 3, 4}
print(a & b)   # 交集：{2, 3}
print(a - b)   # 差集：{1}
```

### 4.4 元组（Tuple）

元组是不可变的有序集合。

```python
# 创建元组
coordinates = (10, 20)
point = (1, 2, 3)

# 元组不可修改
# coordinates[0] = 5  # 会报错

# 元组解包
x, y = coordinates
print(x)  # 10
print(y)  # 20

# 单元素元组需要逗号
single = (42,)
```

### 6.2 参数类型

```python
# 默认参数
def greet(name="World"):
    print(f"Hello, {name}!")

greet()         # Hello, World!
greet("Alice")  # Hello, Alice!

# 关键字参数
def create_profile(name, age, city):
    return {"name": name, "age": age, "city": city}

profile = create_profile(age=25, city="Beijing", name="Alice")

# 可变参数
def sum_all(*numbers):
    return sum(numbers)

print(sum_all(1, 2, 3, 4))  # 10

# 关键字可变参数
def create_dict(**kwargs):
    return kwargs

print(create_dict(name="Alice", age=25))
# {'name': 'Alice', 'age': 25}
```

### 6.3 Lambda表达式

```python
# Lambda是简短的匿名函数
square = lambda x: x ** 2
print(square(5))  # 25

# 在函数中使用
numbers = [1, 2, 3, 4, 5]
squared = list(map(lambda x: x ** 2, numbers))
print(squared)  # [1, 4, 9, 16, 25]

# 过滤
evens = list(filter(lambda x: x % 2 == 0, numbers))
print(evens)  # [2, 4]
```

## 八、异步编程（重点）

**异步编程是AI应用开发的核心技能**，因为：
- LLM API调用需要等待响应
- Agent可能同时处理多个任务
- 流式响应需要异步处理

### 8.1 基本概念

**同步 vs 异步**：

```python
# 同步执行（顺序执行）
def task1():
    time.sleep(1)  # 假设需要1秒
    return "任务1完成"

def task2():
    time.sleep(1)
    return "任务2完成"

# 同步调用：总共需要2秒
result1 = task1()  # 等1秒
result2 = task2()  # 再等1秒

# 异步执行（并发执行）
import asyncio

async def async_task1():
    await asyncio.sleep(1)
    return "任务1完成"

async def async_task2():
    await asyncio.sleep(1)
    return "任务2完成"

# 异步调用：总共只需要1秒（两个任务并发）
async def main():
    results = await asyncio.gather(
        async_task1(),
        async_task2()
    )
    print(results)

asyncio.run(main())
```

### 8.2 async/await语法

```python
import asyncio

# 定义异步函数
async def fetch_data(url):
    print(f"开始获取 {url}")
    await asyncio.sleep(1)  # 模拟网络请求
    print(f"完成获取 {url}")
    return f"数据来自 {url}"

# 调用异步函数
async def main():
    # 单个异步调用
    result = await fetch_data("https://api.example.com")
    print(result)

asyncio.run(main())
```

### 8.3 并发执行

```python
import asyncio

async def fetch_data(url, delay):
    await asyncio.sleep(delay)
    return f"数据来自 {url}"

async def main():
    # 使用gather并发执行多个任务
    results = await asyncio.gather(
        fetch_data("url1", 1),
        fetch_data("url2", 2),
        fetch_data("url3", 1.5)
    )
    print(results)
    # 三个任务并发执行，总共只需2秒

asyncio.run(main())
```

### 8.4 实际应用示例（调用OpenAI API）

```python
import asyncio
from openai import AsyncOpenAI

# 创建异步客户端
client = AsyncOpenAI()

async def chat(message):
    # 异步调用API
    response = await client.chat.completions.create(
        model="gpt-3.5-turbo",
        messages=[{"role": "user", "content": message}]
    )
    return response.choices[0].message.content

async def main():
    # 并发调用多个请求
    messages = ["问题1", "问题2", "问题3"]
    tasks = [chat(msg) for msg in messages]
    results = await asyncio.gather(*tasks)

    for result in results:
        print(result)

asyncio.run(main())
```

### 8.5 异步生成器（Streaming）

```python
import asyncio
from openai import AsyncOpenAI

client = AsyncOpenAI()

async def stream_chat(message):
    # 流式响应
    stream = await client.chat.completions.create(
        model="gpt-3.5-turbo",
        messages=[{"role": "user", "content": message}],
        stream=True
    )

    # 异步迭代流式响应
    async for chunk in stream:
        if chunk.choices[0].delta.content:
            print(chunk.choices[0].delta.content, end="", flush=True)

async def main():
    await stream_chat("请写一首短诗")

asyncio.run(main())
```
