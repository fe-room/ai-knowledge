# 1.1.2 JavaScript/TypeScript入门

> JavaScript是全栈开发的核心语言，TypeScript提供了类型安全

## 一、JavaScript简介与应用领域

### 什么是JavaScript？

JavaScript是一门动态类型的编程语言，最初用于浏览器端，现在通过Node.js也能在服务器端运行。

**JavaScript的特点**：
- ✅ 动态类型，灵活易用
- ✅ 异步编程天然支持（Promise、async/await）
- ✅ 丰富的生态系统（npm）
- ✅ 全栈应用能力（前端+后端）

**JavaScript在AI领域的应用**：
- 前端AI应用开发（React、Vue、Angular）
- 全栈AI应用（Next.js、Express）
- Node.js后端集成LLM API
- LangChain.js框架（Agent开发）
- 浏览器端AI推理

## 三、JavaScript基本语法（ES6+）

### 3.1 变量声明

ES6引入了`let`和`const`，推荐使用它们代替`var`。

```javascript
// let - 可变变量
let count = 0;
count = 1;  // 可以修改

// const - 常量（推荐用于不变的值）
const PI = 3.14159;
// PI = 3.14  // 会报错

// var - 不推荐使用（存在作用域问题）
var oldWay = "不推荐";

// 最佳实践：优先使用const，需要修改时使用let
```

### 3.2 数据类型

```javascript
// 基本类型
let string = "Hello";
let number = 42;
let float = 3.14;
let boolean = true;
let nullValue = null;
let undefinedValue = undefined;

// 对象和数组
let obj = {
  name: "Alice",
  age: 25
};

let arr = [1, 2, 3, 4, 5];

// Symbol（ES6新增）
let sym = Symbol("unique");

// 类型检查
console.log(typeof string);    // "string"
console.log(typeof number);     // "number"
console.log(typeof obj);        // "object"
console.log(typeof arr);        // "object"（数组也是对象）
console.log(Array.isArray(arr)); // true（正确判断数组）
```

### 3.3 箭头函数

箭头函数是ES6的重要特性，语法简洁。

```javascript
// 传统函数
function add(a, b) {
  return a + b;
}

// 箭头函数
const addArrow = (a, b) => a + b;

// 多行箭头函数
const greet = (name) => {
  const message = `Hello, ${name}`;
  return message;
};

// 单参数可省略括号
const square = x => x * x;

// 无参数
const sayHi = () => console.log("Hi!");

// 对比
console.log(add(3, 5));      // 8
console.log(addArrow(3, 5)); // 8
```

### 3.4 模板字符串

```javascript
// 传统拼接
const name = "Alice";
const greeting = "Hello, " + name + "!";

// 模板字符串（推荐）
const greetingTemplate = `Hello, ${name}!`;

// 多行字符串
const multiLine = `
第一行
第二行
第三行
`;

// 表达式嵌入
const a = 10;
const b = 20;
const result = `a + b = ${a + b}`;  // "a + b = 30"
```

### 3.5 解构赋值

```javascript
// 数组解构
const arr = [1, 2, 3];
const [a, b, c] = arr;
console.log(a, b, c);  // 1, 2, 3

// 对象解构
const person = { name: "Alice", age: 25 };
const { name, age } = person;
console.log(name, age);  // "Alice", 25

// 重命名
const { name: userName } = person;
console.log(userName);  // "Alice"

// 默认值
const { job = "Developer" } = person;
console.log(job);  // "Developer"

// 函数参数解构
function greet({ name, age }) {
  console.log(`${name} is ${age} years old`);
}
greet(person);
```

### 3.6 扩展运算符

```javascript
// 数组扩展
const arr1 = [1, 2, 3];
const arr2 = [4, 5, 6];
const combined = [...arr1, ...arr2];  // [1, 2, 3, 4, 5, 6]

// 复制数组
const copied = [...arr1];

// 对象扩展
const obj1 = { a: 1, b: 2 };
const obj2 = { c: 3, ...obj1 };  // { c: 3, a: 1, b: 2 }

// 函数参数
function sum(...numbers) {
  return numbers.reduce((a, b) => a + b, 0);
}
console.log(sum(1, 2, 3, 4));  // 10
```

### 3.7 数组方法

```javascript
const numbers = [1, 2, 3, 4, 5];

// map - 映射
const doubled = numbers.map(x => x * 2);
console.log(doubled);  // [2, 4, 6, 8, 10]

// filter - 过滤
const evens = numbers.filter(x => x % 2 === 0);
console.log(evens);  // [2, 4]

// reduce - 归约
const sum = numbers.reduce((acc, x) => acc + x, 0);
console.log(sum);  // 15

// find - 查找
const found = numbers.find(x => x > 3);
console.log(found);  // 4

// forEach - 遍历
numbers.forEach(x => console.log(x));

// includes - 包含检查
console.log(numbers.includes(3));  // true

// 链式调用
const result = numbers
  .filter(x => x % 2 === 0)
  .map(x => x * 10);
console.log(result);  // [20, 40]
```

## 四、TypeScript基础

### 4.1 为什么学习TypeScript？

**TypeScript的优势**：
- ✅ 类型安全，减少运行时错误
- ✅ 更好的IDE支持（智能提示）
- ✅ 代码更易维护和重构
- ✅ 大型项目必备

### 4.2 TypeScript安装

```bash
# 安装TypeScript
npm install -g typescript

# 或作为项目依赖
npm install --save-dev typescript

# 验证安装
tsc --version

# 编译TypeScript
tsc filename.ts
```

### 4.3 基本类型

```typescript
// 基本类型
let str: string = "Hello";
let num: number = 42;
let bool: boolean = true;

// 数组类型
let arr1: number[] = [1, 2, 3];
let arr2: Array<number> = [1, 2, 3];  // 泛型语法

// 对象类型
let obj: { name: string; age: number } = {
  name: "Alice",
  age: 25
};

// 元组
let tuple: [string, number] = ["Alice", 25];

// 枚举
enum Color {
  Red,
  Green,
  Blue
}
let color: Color = Color.Red;

// Any（任意类型，尽量避免使用）
let anything: any = "可以是任何类型";
anything = 42;

// Unknown（类型安全的any）
let unknown: unknown = "未知类型";
// unknown.toFixed()  // 错误，需要类型检查
if (typeof unknown === "number") {
  unknown.toFixed();  // 正确
}

// Null和Undefined
let nullable: string | null = null;
let undefinable: string | undefined = undefined;
```

### 4.4 函数类型

```typescript
// 函数类型定义
function add(a: number, b: number): number {
  return a + b;
}

// 箭头函数
const multiply = (a: number, b: number): number => a * b;

// 可选参数
function greet(name: string, greeting?: string): string {
  return greeting ? `${greeting}, ${name}` : `Hello, ${name}`;
}

// 默认参数
function greetWithDefault(name: string, greeting: string = "Hello"): string {
  return `${greeting}, ${name}`;
}

// 函数类型别名
type MathOperation = (a: number, b: number) => number;

const subtract: MathOperation = (a, b) => a - b;
```

### 4.5 接口

```typescript
// 定义接口
interface Person {
  name: string;
  age: number;
  greet(): string;
}

// 实现接口
class Student implements Person {
  constructor(public name: string, public age: number) {}

  greet(): string {
    return `Hello, I'm ${this.name}`;
  }
}

// 接口扩展
interface Employee extends Person {
  job: string;
  salary: number;
}
```

## 五、异步编程（重点）

**异步编程是JavaScript的核心特性**，对AI应用开发至关重要。

### 5.1 Callback（传统方式）

```javascript
// 回调函数（不推荐，容易造成回调地狱）
function fetchData(callback) {
  setTimeout(() => {
    callback("数据");
  }, 1000);
}

fetchData((data) => {
  console.log(data);
});
```

### 5.2 Promise

Promise是ES6引入的异步解决方案。

```javascript
// 创建Promise
const promise = new Promise((resolve, reject) => {
  setTimeout(() => {
    const success = true;
    if (success) {
      resolve("成功数据");
    } else {
      reject("失败原因");
    }
  }, 1000);
});

// 使用Promise
promise
  .then(data => {
    console.log("成功:", data);
    return data + " 处理后";
  })
  .then(processed => {
    console.log("处理后:", processed);
  })
  .catch(error => {
    console.error("失败:", error);
  })
  .finally(() => {
    console.log("完成");
  });

// Promise链式调用
fetchData1()
  .then(data1 => fetchData2(data1))
  .then(data2 => fetchData3(data2))
  .catch(error => console.error(error));
```

### 5.3 async/await（推荐）

async/await是ES8引入的语法，让异步代码看起来像同步代码。

```javascript
// async函数
async function fetchData() {
  // await等待Promise完成
  const data = await fetchFromAPI();
  console.log(data);
  return data;
}

// 实际应用示例
async function fetchMultipleAPIs() {
  try {
    // 并发请求
    const [api1Data, api2Data] = await Promise.all([
      fetchFromAPI1(),
      fetchFromAPI2()
    ]);

    console.log("API1:", api1Data);
    console.log("API2:", api2Data);

    return { api1Data, api2Data };
  } catch (error) {
    console.error("请求失败:", error);
  }
}

// 调用async函数
fetchMultipleAPIs();
```

### 5.4 Promise.all和并发控制

```javascript
// Promise.all - 等待所有完成
const promises = [
  fetchFromAPI1(),
  fetchFromAPI2(),
  fetchFromAPI3()
];

// 如果所有Promise都成功，返回所有结果
const results = await Promise.all(promises);

// 如果有一个失败，整个失败
// 使用Promise.allSettled获取所有结果（包括失败的）
const settled = await Promise.allSettled(promises);
// settled包含每个Promise的状态和结果

// Promise.race - 返回最先完成的
const fastest = await Promise.race(promises);
```

### 5.5 模拟AI API调用

```javascript
// 模拟调用OpenAI API
const openai = require('openai');

async function callOpenAI(prompt) {
  try {
    const response = await openai.chat.completions.create({
      model: "gpt-3.5-turbo",
      messages: [{ role: "user", content: prompt }]
    });

    return response.choices[0].message.content;
  } catch (error) {
    console.error("API调用失败:", error);
    throw error;
  }
}

// 并发调用多个API
async function batchCallAI(prompts) {
  const promises = prompts.map(prompt => callOpenAI(prompt));
  const results = await Promise.all(promises);
  return results;
}

// 流式响应
async function streamAI(prompt) {
  const stream = await openai.chat.completions.create({
    model: "gpt-3.5-turbo",
    messages: [{ role: "user", content: prompt }],
    stream: true
  });

  for await (const chunk of stream) {
    const content = chunk.choices[0]?.delta?.content || "";
    console.log(content);
  }
}
```

## 六、模块系统

### 6.1 ES Modules（推荐）

```javascript
// 导出模块 (module.js)
export const name = "Alice";

export function greet(name) {
  return `Hello, ${name}`;
}

export class Person {
  constructor(name) {
    this.name = name;
  }
}

// 默认导出
export default function mainFunction() {
  console.log("主函数");
}

// 导入模块 (main.js)
import { name, greet, Person } from './module.js';
import mainFunction from './module.js';

// 导入所有
import * as module from './module.js';

// 重命名导入
import { greet as hello } from './module.js';
```

### 6.2 CommonJS（Node.js传统）

```javascript
// 导出 (module.js)
const name = "Alice";

function greet(name) {
  return `Hello, ${name}`;
}

module.exports = {
  name,
  greet
};

// 导入 (main.js)
const module = require('./module.js');
console.log(module.name);
module.greet("Bob");
```
