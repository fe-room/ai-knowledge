# 2.3.2 ReAct Agent模式

> 掌握Reasoning + Acting的Agent设计模式

## 一、ReAct模式介绍

### 1.1 什么是ReAct？

**ReAct = Reasoning + Acting**

一种结合推理（Reasoning）和行动（Acting）的Agent设计模式。

```
传统模式：
用户问题 → LLM → 直接回答（可能错误）

ReAct模式：
用户问题 → LLM思考 → 执行动作 → 观察结果 → 继续思考 → 最终回答
```

### 1.2 ReAct流程

```
┌────────────────────────────────────────────┐
│              ReAct循环                      │
├────────────────────────────────────────────┤
│                                            │
│    用户问题                                 │
│        │                                   │
│        ↓                                   │
│    ┌─────────┐                             │
│    │ Thought │ ← 思考下一步做什么           │
│    └─────────┘                             │
│        │                                   │
│        ↓                                   │
│    ┌─────────┐                             │
│    │ Action  │ ← 选择并执行动作             │
│    └─────────┘                             │
│        │                                   │
│        ↓                                   │
│    ┌─────────┐                             │
│    │Observation│ ← 观察执行结果             │
│    └─────────┘                             │
│        │                                   │
│        ↓                                   │
│    ┌─────────┐                             │
│    │ Thought │ ← 继续思考                   │
│    └─────────┘                             │
│        │                                   │
│        ↓                                   │
│    ... 循环直到解决问题                      │
│        │                                   │
│        ↓                                   │
│    ┌─────────┐                             │
│    │ Answer  │ ← 最终答案                   │
│    └─────────┘                             │
│                                            │
└────────────────────────────────────────────┘
```

## 二、ReAct实现

### 2.1 基础ReAct Agent

```python
"""
基础ReAct Agent实现
"""

from openai import OpenAI
import json

class ReActAgent:
    """ReAct Agent"""

    def __init__(self):
        self.client = OpenAI()
        self.tools = {}
        self.max_iterations = 10

    def register_tool(self, name, function, description):
        """注册工具"""
        self.tools[name] = {
            "function": function,
            "description": description
        }

    def run(self, question):
        """运行Agent"""
        history = []
        iteration = 0

        while iteration < self.max_iterations:
            iteration += 1

            # 构建提示词
            prompt = self._build_prompt(question, history)

            # 获取LLM响应
            response = self.client.chat.completions.create(
                model="gpt-3.5-turbo",
                messages=[{"role": "user", "content": prompt}]
            )

            output = response.choices[0].message.content

            # 解析输出
            thought, action, action_input = self._parse_output(output)

            # 记录思考
            history.append(f"Thought: {thought}")

            # 检查是否完成
            if action == "Finish":
                return action_input

            # 执行动作
            if action in self.tools:
                result = self.tools[action]["function"](action_input)
                history.append(f"Action: {action}[{action_input}]")
                history.append(f"Observation: {result}")
            else:
                history.append(f"Action: {action}[{action_input}]")
                history.append(f"Observation: Unknown action")

        return "达到最大迭代次数"

    def _build_prompt(self, question, history):
        """构建提示词"""
        tools_desc = "\n".join([
            f"{name}: {desc}"
            for name, desc in [(k, v["description"]) for k, v in self.tools.items()]
        ])

        prompt = f"""
你需要回答问题。你可以使用以下工具：

{tools_desc}

使用以下格式：

Thought: 思考下一步
Action: 工具名称
Action Input: 工具输入
Observation: 工具输出
... (重复Thought/Action/Observation)
Thought: 我知道答案了
Action: Finish
Action Input: 最终答案

问题: {question}

{chr(10).join(history) if history else ''}
"""
        return prompt

    def _parse_output(self, output):
        """解析输出"""
        thought = ""
        action = ""
        action_input = ""

        lines = output.split("\n")
        for line in lines:
            if line.startswith("Thought:"):
                thought = line.replace("Thought:", "").strip()
            elif line.startswith("Action:"):
                action = line.replace("Action:", "").strip()
            elif line.startswith("Action Input:"):
                action_input = line.replace("Action Input:", "").strip()

        return thought, action, action_input

# 使用示例
agent = ReActAgent()

# 注册工具
agent.register_tool(
    "search",
    lambda query: f"搜索结果: {query}",
    "搜索互联网获取信息"
)

agent.register_tool(
    "calculate",
    lambda expr: str(eval(expr)),
    "执行数学计算"
)

# 运行
result = agent.run("北京的人口是多少？")
print(result)
```

### 2.2 使用Function Calling

```python
class ReActWithTools:
    """使用Function Calling的ReAct"""

    def __init__(self):
        self.client = OpenAI()
        self.tools = []
        self.functions = {}

    def register_tool(self, name, function, parameters):
        """注册工具"""
        self.tools.append({
            "type": "function",
            "function": {
                "name": name,
                "description": function.__doc__ or name,
                "parameters": parameters
            }
        })
        self.functions[name] = function

    def run(self, question):
        """运行"""
        messages = [{"role": "user", "content": question}]

        for _ in range(10):
            response = self.client.chat.completions.create(
                model="gpt-3.5-turbo",
                messages=messages,
                tools=self.tools
            )

            message = response.choices[0].message
            messages.append(message)

            if not message.tool_calls:
                return message.content

            # 执行工具
            for tool_call in message.tool_calls:
                func_name = tool_call.function.name
                func_args = json.loads(tool_call.function.arguments)

                print(f"[Action] {func_name}({func_args})")

                result = self.functions[func_name](**func_args)
                print(f"[Observation] {result}")

                messages.append({
                    "role": "tool",
                    "tool_call_id": tool_call.id,
                    "content": str(result)
                })

        return "达到最大迭代次数"
```

## 三、ReAct应用场景

### 3.1 信息检索任务

```python
"""
场景：需要多步搜索的问题
"""

def search_wikipedia(topic):
    """搜索维基百科"""
    return f"关于{topic}的维基百科信息..."

def search_news(keyword):
    """搜索新闻"""
    return f"关于{keyword}的新闻..."

agent = ReActAgent()
agent.register_tool("wikipedia", search_wikipedia, "搜索维基百科")
agent.register_tool("news", search_news, "搜索新闻")

# 问题："2024年诺贝尔物理学奖得主是谁？他们的主要贡献是什么？"
# Agent流程：
# Thought: 首先搜索诺贝尔物理学奖2024年的新闻
# Action: news["诺贝尔物理学奖 2024"]
# Observation: [新闻内容]
# Thought: 我知道得主是谁了，现在搜索他们的贡献
# Action: wikipedia["得主姓名"]
# Observation: [维基百科内容]
# Thought: 我现在可以回答问题了
# Action: Finish["完整答案"]
```

### 3.2 计算任务

```python
"""
场景：需要多步计算的问题
"""

def calculate(expression):
    """数学计算"""
    try:
        return str(eval(expression))
    except:
        return "计算错误"

def convert(value, from_unit, to_unit):
    """单位转换"""
    # 简化示例
    conversions = {
        ("km", "mile"): 0.621371,
        ("mile", "km"): 1.60934,
    }
    ratio = conversions.get((from_unit, to_unit), 1)
    return str(float(value) * ratio)

agent = ReActAgent()
agent.register_tool("calculate", calculate, "数学计算")
agent.register_tool("convert", convert, "单位转换")

# 问题："100公里等于多少英里？如果跑完需要2小时，平均速度是多少英里/小时？"
# Agent流程：
# Thought: 先转换距离单位
# Action: convert["100", "km", "mile"]
# Observation: 62.1371
# Thought: 计算平均速度
# Action: calculate["62.1371 / 2"]
# Observation: 31.06855
# Thought: 我知道答案了
# Action: Finish["100公里=62.14英里，平均速度31.07英里/小时"]
```

### 4.2 错误处理

```python
class RobustReActAgent(ReActAgent):
    """健壮的ReAct Agent"""

    def run(self, question):
        """运行，带错误处理"""
        history = []

        for i in range(self.max_iterations):
            try:
                prompt = self._build_prompt(question, history)
                response = self.client.chat.completions.create(
                    model="gpt-3.5-turbo",
                    messages=[{"role": "user", "content": prompt}]
                )

                output = response.choices[0].message.content

                thought, action, action_input = self._parse_output(output)

                if not thought or not action:
                    history.append("Observation: 格式错误，请按正确格式输出")
                    continue

                if action == "Finish":
                    return action_input

                if action not in self.tools:
                    history.append(f"Observation: 工具'{action}'不存在")
                    continue

                try:
                    result = self.tools[action]["function"](action_input)
                except Exception as e:
                    result = f"工具执行错误: {str(e)}"

                history.append(f"Thought: {thought}")
                history.append(f"Action: {action}[{action_input}]")
                history.append(f"Observation: {result}")

            except Exception as e:
                history.append(f"Observation: 系统错误: {str(e)}")

        return "抱歉，我无法完成这个任务"
```

## 五、ReAct vs 其他模式

### 5.1 对比

| 特性 | ReAct | Plan-Execute | 单步调用 |
|------|-------|--------------|---------|
| 复杂度 | 中等 | 较高 | 低 |
| 适用任务 | 多步推理 | 复杂任务 | 简单查询 |
| 灵活性 | 高 | 中 | 低 |
| 执行效率 | 中 | 高 | 高 |
| 可解释性 | 高 | 中 | 低 |

### 5.2 选择建议

```
选择ReAct：
- ✅ 需要逐步推理
- ✅ 任务不确定性高
- ✅ 需要观察反馈调整

选择Plan-Execute：
- ✅ 任务步骤明确
- ✅ 可以预先规划
- ✅ 需要优化执行顺序

选择单步调用：
- ✅ 简单查询
- ✅ 工具调用明确
- ✅ 无需多步推理
```

## 总结

### 核心要点

1. **ReAct** = Reasoning + Acting
2. **循环流程**：Thought → Action → Observation → 循环
3. **实现方式**：手动解析或Function Calling
4. **适用场景**：信息检索、多步计算
5. **优化方向**：提示词设计、错误处理
