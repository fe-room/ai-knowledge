# 2.3.1 Agent基础架构

> 理解Agent的核心组件，构建第一个简单Agent

## 一、Agent核心组件

### 1.1 四大核心组件

Agent由四个核心组件组成：

```
┌────────────────────────────────────────────┐
│              Agent核心组件                  │
├────────────────────────────────────────────┤
│                                            │
│  1. 大脑（Brain/LLM）                      │
│     - 理解用户意图                         │
│     - 规划行动步骤                         │
│     - 决策下一步                           │
│                                            │
│  2. 感知（Perception）                     │
│     - 接收用户输入                         │
│     - 解析环境信息                         │
│     - 理解工具返回                         │
│                                            │
│  3. 记忆（Memory）                         │
│     - 短期记忆：对话历史                   │
│     - 长期记忆：知识存储                   │
│     - 工作记忆：当前状态                   │
│                                            │
│  4. 工具（Tools）                          │
│     - 搜索工具                             │
│     - 数据库查询                           │
│     - API调用                              │
│     - 自定义工具                           │
│                                            │
└────────────────────────────────────────────┘
```

### 1.2 组件详解

**1. 大脑（LLM）**

大脑是Agent的核心，负责：
- 理解用户意图
- 规划任务步骤
- 决策使用哪个工具
- 生成最终回复

```python
# 大脑示例（使用LLM）
def think(self, user_input, context):
    """思考：理解意图并规划"""
    prompt = f"""
    用户输入：{user_input}
    上下文：{context}

    请分析用户意图，并规划下一步行动。
    """

    response = self.llm.generate(prompt)
    return self.parse_thought(response)
```

**2. 感知（Perception）**

感知负责信息输入：
- 解析用户输入
- 接收工具返回结果
- 理解环境状态

```python
def perceive(self, input_data):
    """感知：解析输入"""
    # 解析用户意图
    intent = self.parse_intent(input_data)

    # 提取关键实体
    entities = self.extract_entities(input_data)

    # 获取上下文
    context = self.memory.get_context()

    return {
        "intent": intent,
        "entities": entities,
        "context": context
    }
```

**3. 记忆（Memory）**

记忆管理系统状态：
- 对话历史
- 任务进度
- 重要信息

```python
class Memory:
    def __init__(self):
        self.short_term = []  # 短期记忆
        self.long_term = {}   # 长期记忆

    def add_message(self, role, content):
        """添加消息到短期记忆"""
        self.short_term.append({
            "role": role,
            "content": content
        })

    def save_to_long_term(self, key, value):
        """保存到长期记忆"""
        self.long_term[key] = value

    def get_context(self):
        """获取上下文"""
        return self.short_term[-10:]  # 最近10条
```

**4. 工具（Tools）**

工具扩展Agent能力：
- 搜索引擎
- 数据库
- API
- 代码执行

```python
class Tool:
    def __init__(self, name, description, func):
        self.name = name
        self.description = description
        self.func = func

    def execute(self, *args, **kwargs):
        """执行工具"""
        return self.func(*args, **kwargs)

# 定义工具
search_tool = Tool(
    name="search",
    description="搜索互联网信息",
    func=search_engine
)

calculator_tool = Tool(
    name="calculator",
    description="执行数学计算",
    func=calculate
)
```

## 二、Agent工作流程

### 2.1 基本工作流程

```
用户输入
   │
   ↓
┌─────────────┐
│    感知     │ ← 解析输入，提取信息
└─────┬───────┘
      │
      ↓
┌─────────────┐
│    思考     │ ← 理解意图，规划步骤
└─────┬───────┘
      │
      ↓
┌─────────────┐     否
│ 任务完成？   │ ───────────┐
└─────┬───────┘            │
      │ 是                 │
      ↓                    ↓
┌─────────────┐     ┌─────────────┐
│  生成回复   │     │    执行     │ ← 调用工具
└─────────────┘     └─────┬───────┘
                          │
                          ↓
                    ┌─────────────┐
                    │    观察     │ ← 获取结果
                    └─────┬───────┘
                          │
                          └──────────→ 回到思考
```

### 2.2 工作流程实现

```python
class SimpleAgent:
    def __init__(self, llm, tools):
        self.llm = llm
        self.tools = {tool.name: tool for tool in tools}
        self.memory = Memory()

    def run(self, user_input):
        """Agent主循环"""
        # 感知
        perception = self.perceive(user_input)

        # 记录用户输入
        self.memory.add_message("user", user_input)

        # 主循环
        while True:
            # 思考
            thought = self.think(perception)

            # 判断是否需要执行动作
            if thought["action"] == "finish":
                # 任务完成，生成回复
                response = self.generate_response()
                self.memory.add_message("assistant", response)
                return response

            # 执行动作
            result = self.act(thought["action"], thought["parameters"])

            # 观察
            observation = self.observe(result)

            # 更新感知
            perception["last_result"] = observation

    def perceive(self, input_data):
        """感知阶段"""
        return {
            "user_input": input_data,
            "memory": self.memory.get_context()
        }

    def think(self, perception):
        """思考阶段"""
        prompt = f"""
        用户输入：{perception['user_input']}
        记忆：{perception['memory']}

        请决定下一步：
        1. 如果需要更多信息，返回 {{"action": "search", "parameters": {{"query": "..."}}}}
        2. 如果可以回答，返回 {{"action": "finish", "response": "..."}}
        """

        response = self.llm.generate(prompt)
        return self.parse_response(response)

    def act(self, action, parameters):
        """执行阶段"""
        if action in self.tools:
            return self.tools[action].execute(**parameters)
        return None

    def observe(self, result):
        """观察阶段"""
        return result

    def generate_response(self):
        """生成最终回复"""
        prompt = "根据收集的信息，生成最终回复..."
        return self.llm.generate(prompt)
```

## 三、简单Agent实现

### 3.1 完整代码示例

```python
"""
简单Agent实现示例
功能：能够回答问题和搜索信息
"""

import os
from openai import OpenAI
from dotenv import load_dotenv

load_dotenv()

class SimpleAgent:
    """简单Agent类"""

    def __init__(self):
        # 初始化LLM
        self.llm = OpenAI(api_key=os.getenv("OPENAI_API_KEY"))

        # 消息历史
        self.messages = []

        # 定义工具
        self.tools = {
            "search": self.search_tool,
            "calculator": self.calculator_tool
        }

    def search_tool(self, query):
        """模拟搜索工具"""
        # 实际项目中应该调用真实的搜索API
        return f"[搜索结果] 关于'{query}'的信息..."

    def calculator_tool(self, expression):
        """计算器工具"""
        try:
            result = eval(expression)
            return f"[计算结果] {expression} = {result}"
        except Exception as e:
            return f"[计算错误] {str(e)}"

    def think(self, user_input):
        """思考阶段：决定下一步行动"""
        prompt = f"""
        你是一个智能助手，可以使用以下工具：
        - search: 搜索信息
        - calculator: 执行数学计算

        用户输入：{user_input}

        请决定：
        1. 是否需要使用工具？（如果需要，返回JSON）
        2. 如果可以直接回答，返回 {{"action": "answer", "response": "..."}}

        只返回JSON，不要其他内容。
        """

        response = self.llm.chat.completions.create(
            model="gpt-3.5-turbo",
            messages=[{"role": "user", "content": prompt}],
            temperature=0
        )

        return response.choices[0].message.content

    def run(self, user_input):
        """运行Agent"""
        print(f"\n用户: {user_input}")

        # 添加用户消息
        self.messages.append({"role": "user", "content": user_input})

        # 思考
        print("[Agent思考中...]")
        thought = self.think(user_input)

        # 解析思考结果
        import json
        try:
            decision = json.loads(thought)
        except:
            decision = {"action": "answer", "response": thought}

        # 执行
        if decision["action"] == "answer":
            response = decision["response"]
        elif decision["action"] == "search":
            result = self.search_tool(decision.get("query", ""))
            # 基于搜索结果生成回答
            response = self.generate_final_answer(user_input, result)
        elif decision["action"] == "calculator":
            result = self.calculator_tool(decision.get("expression", ""))
            response = result
        else:
            response = "我无法处理这个请求。"

        # 记录回复
        self.messages.append({"role": "assistant", "content": response})

        print(f"Agent: {response}")
        return response

    def generate_final_answer(self, question, search_result):
        """基于搜索结果生成最终答案"""
        prompt = f"""
        问题：{question}
        搜索结果：{search_result}

        请基于搜索结果回答问题。
        """

        response = self.llm.chat.completions.create(
            model="gpt-3.5-turbo",
            messages=[{"role": "user", "content": prompt}]
        )

        return response.choices[0].message.content

# 使用示例
if __name__ == "__main__":
    agent = SimpleAgent()

    # 测试对话
    agent.run("你好！")
    agent.run("帮我计算 123 * 456")
    agent.run("搜索Python是什么")
```

## 四、Agent设计模式

### 4.1 单Agent模式

最简单的模式，一个Agent处理所有任务。

```
用户 → Agent → 回复
```

**适用场景**：
- 简单问答
- 单一任务
- 原型开发

### 4.2 多Agent模式

多个Agent协作完成复杂任务。

```
用户 → 主Agent → 子Agent1
                → 子Agent2
                → 子Agent3
             ↓
         汇总结果
```

**适用场景**：
- 复杂任务分解
- 需要专业分工
- 大型项目

### 4.3 层级Agent模式

主Agent管理子Agent。

```
       主Agent
      /   |   \
   Agent1 Agent2 Agent3
    /  \     |     |
 Worker  Worker  Worker
```

**适用场景**：
- 企业级应用
- 复杂工作流
- 需要严格管理

## 五、Agent vs 传统应用

### 5.1 对比分析

| 特性 | 传统应用 | Agent应用 |
|------|---------|----------|
| 决策 | 硬编码规则 | LLM自主决策 |
| 灵活性 | 低 | 高 |
| 可扩展性 | 需要重新编码 | 添加工具即可 |
| 错误处理 | 预定义 | 自适应 |
| 复杂任务 | 需要大量代码 | 自动分解执行 |

### 5.2 选择建议

**使用传统应用**：
- 确定性任务
- 规则明确
- 性能要求高

**使用Agent**：
- 需要灵活性
- 任务复杂多变
- 需要自主决策

## 六、最佳实践

### 6.1 设计原则

1. **单一职责**：每个Agent专注一类任务
2. **工具明确**：工具定义要清晰
3. **错误处理**：优雅处理失败情况
4. **可观测性**：记录Agent决策过程

### 6.2 常见陷阱

1. **过度复杂**：简单任务不要用Agent
2. **工具过多**：工具过多导致选择困难
3. **无限制循环**：设置最大循环次数
4. **忽略安全**：限制敏感操作
