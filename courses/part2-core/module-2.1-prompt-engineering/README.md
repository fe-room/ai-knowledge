## 核心概念

**Prompt Engineering是什么？**

Prompt Engineering（提示词工程）是设计和优化输入给LLM的提示词，以获得期望输出的技术和实践。

```
提示词（输入） → LLM → 输出
     ↑                  ↓
     └──── 反馈优化 ←───┘
```

**为什么重要？**
- 提示词质量直接决定输出质量
- 相同模型，不同提示词可能得到截然不同的结果
- 是AI应用开发的核心技能

## 实践建议

1. **多实践**：每个技术都要动手尝试
2. **对比效果**：尝试不同提示词，对比输出
3. **建立库**：收集有效的提示词模板
4. **迭代优化**：持续改进提示词
