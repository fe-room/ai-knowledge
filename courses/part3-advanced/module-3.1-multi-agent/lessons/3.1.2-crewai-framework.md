# 3.1.2 CrewAI框架实战

> 使用CrewAI构建多Agent协作系统

## 一、CrewAI概述

### 1.1 什么是CrewAI？

**CrewAI** 是一个多Agent协作框架，让多个AI Agent像团队一样工作。

```
CrewAI核心概念：

Agent（代理）
  └── 执行特定角色的AI实体

Task（任务）
  └── Agent需要完成的工作

Crew（团队）
  └── Agent和Task的组合

Process（流程）
  └── 任务执行的方式
```

### 1.2 安装

```python
pip install crewai
pip install crewai-tools
```

## 二、Agent定义

### 2.1 创建Agent

```python
"""
CrewAI Agent定义
"""

from crewai import Agent
from langchain_openai import ChatOpenAI

# 创建LLM
llm = ChatOpenAI(model="gpt-4", temperature=0.7)

# 定义研究员Agent
researcher = Agent(
    role="高级研究员",
    goal="研究和分析最新AI发展趋势",
    backstory="""
    你是一位经验丰富的研究员，擅长收集和分析信息。
    你有深厚的学术背景，能够快速理解和总结复杂的概念。
    """,
    verbose=True,
    allow_delegation=False,
    llm=llm
)

# 定义作家Agent
writer = Agent(
    role="内容创作者",
    goal="创作引人入胜且信息丰富的内容",
    backstory="""
    你是一位才华横溢的作家，擅长将复杂的信息转化为易懂的文章。
    你有独特的写作风格，能够吸引读者的注意力。
    """,
    verbose=True,
    allow_delegation=True,  # 可以委托任务
    llm=llm
)

# 定义编辑Agent
editor = Agent(
    role="内容编辑",
    goal="确保内容质量和一致性",
    backstory="""
    你是一位严谨的编辑，注重细节和质量。
    你擅长发现和修正文章中的问题，确保内容准确流畅。
    """,
    verbose=True,
    llm=llm
)
```

## 三、Task定义

### 3.1 创建Task

```python
"""
CrewAI Task定义
"""

from crewai import Task

# 研究任务
research_task = Task(
    description="""
    研究以下主题的最新发展趋势：
    {topic}

    请收集以下信息：
    1. 主要技术突破
    2. 行业应用案例
    3. 未来发展方向
    4. 潜在挑战

    输出格式：
    - 结构化报告
    - 包含数据来源
    """,
    expected_output="详细的研究报告，包含所有要求的信息",
    agent=researcher,
    tools=[search_tool]  # 可选工具
)

# 写作任务
write_task = Task(
    description="""
    基于以下研究内容，撰写一篇专业文章：

    研究内容：
    {research_result}

    要求：
    - 标题吸引人
    - 结构清晰
    - 语言通俗易懂
    - 字数1500-2000字
    """,
    expected_output="一篇高质量的文章",
    agent=writer,
    context=[research_task],  # 依赖前面的任务
    output_file="article.md"  # 输出文件
)

# 编辑任务
edit_task = Task(
    description="""
    审核并优化以下文章：
    {article}

    检查：
    1. 语法和拼写
    2. 逻辑连贯性
    3. 信息准确性
    4. 可读性
    """,
    expected_output="优化后的文章和修改建议",
    agent=editor,
    context=[write_task]
)
```

### 3.2 任务依赖

```python
"""
任务依赖关系
"""

# 顺序依赖
task1 = Task(description="任务1", agent=agent1)
task2 = Task(description="任务2", agent=agent2, context=[task1])
task3 = Task(description="任务3", agent=agent3, context=[task2])

# 多任务依赖
task_a = Task(description="任务A", agent=agent_a)
task_b = Task(description="任务B", agent=agent_b)
task_c = Task(
    description="任务C（依赖A和B）",
    agent=agent_c,
    context=[task_a, task_b]
)
```

## 四、Crew定义

### 4.1 创建Crew

```python
"""
CrewAI Crew定义
"""

from crewai import Crew, Process

# 创建Crew
crew = Crew(
    agents=[researcher, writer, editor],
    tasks=[research_task, write_task, edit_task],
    process=Process.sequential,  # 顺序执行
    verbose=True
)

# 执行
result = crew.kickoff()
print(result)
```

### 4.2 Process模式

```python
"""
CrewAI执行模式
"""

# 1. 顺序模式（Sequential）
sequential_crew = Crew(
    agents=[agent1, agent2, agent3],
    tasks=[task1, task2, task3],
    process=Process.sequential
)
# 任务按顺序执行，一个完成后再执行下一个

# 2. 层级模式（Hierarchical）
hierarchical_crew = Crew(
    agents=[manager, worker1, worker2, worker3],
    tasks=[main_task],
    process=Process.hierarchical,
    manager_llm=ChatOpenAI(model="gpt-4")
)
# 管理者Agent协调其他Agent执行任务

# 3. 异步执行
async_crew = Crew(
    agents=[agent1, agent2],
    tasks=[task1, task2],
    process=Process.sequential
)
result = await async_crew.kickoff_async()
```

### 5.2 自定义工具

```python
"""
自定义CrewAI工具
"""

from crewai_tools import tool

@tool("数据库查询工具")
def query_database(query: str) -> str:
    """
    查询数据库获取信息

    Args:
        query: SQL查询语句

    Returns:
        查询结果
    """
    # 实现查询逻辑
    result = execute_sql(query)
    return str(result)

@tool("计算器")
def calculate(expression: str) -> float:
    """
    执行数学计算

    Args:
        expression: 数学表达式

    Returns:
        计算结果
    """
    return eval(expression)

# 使用自定义工具
analyst = Agent(
    role="数据分析师",
    goal="分析数据",
    backstory="...",
    tools=[query_database, calculate]
)
```

## 六、完整示例

### 6.1 研究团队

```python
"""
CrewAI完整示例：研究团队
"""

from crewai import Agent, Task, Crew, Process
from crewai_tools import SerperDevTool
from langchain_openai import ChatOpenAI

# 初始化
llm = ChatOpenAI(model="gpt-4")
search_tool = SerperDevTool()

# 定义Agents
researcher = Agent(
    role="研究员",
    goal="收集和分析信息",
    backstory="你是一位专业研究员",
    tools=[search_tool],
    llm=llm,
    verbose=True
)

analyst = Agent(
    role="分析师",
    goal="分析趋势和模式",
    backstory="你是一位数据分析师",
    llm=llm,
    verbose=True
)

writer = Agent(
    role="作家",
    goal="撰写报告",
    backstory="你是一位专业作家",
    llm=llm,
    verbose=True
)

# 定义Tasks
research_task = Task(
    description="研究主题：{topic}",
    expected_output="研究报告",
    agent=researcher
)

analysis_task = Task(
    description="分析研究结果：{research_result}",
    expected_output="分析报告",
    agent=analyst,
    context=[research_task]
)

writing_task = Task(
    description="撰写最终报告",
    expected_output="完整报告",
    agent=writer,
    context=[research_task, analysis_task]
)

# 创建Crew
crew = Crew(
    agents=[researcher, analyst, writer],
    tasks=[research_task, analysis_task, writing_task],
    process=Process.sequential,
    verbose=True
)

# 执行
result = crew.kickoff(inputs={"topic": "2024年AI发展趋势"})
print(result)
```

## 七、高级特性

### 7.1 记忆系统

```python
"""
CrewAI记忆系统
"""

from crewai import Crew
from crewai.memory import LongTermMemory

# 启用记忆
crew = Crew(
    agents=[agent1, agent2],
    tasks=[task1, task2],
    memory=True,  # 启用记忆
    long_term_memory=LongTermMemory()
)

# 记忆会在多次执行间保持
result1 = crew.kickoff(inputs={"topic": "AI"})
result2 = crew.kickoff(inputs={"topic": "ML"})  # 会记住之前的上下文
```

## 总结

### 核心要点

1. **Agent**：定义角色、目标、背景
2. **Task**：定义任务描述和预期输出
3. **Crew**：组合Agent和Task
4. **Process**：顺序或层级执行模式
5. **Tools**：扩展Agent能力

### 最佳实践

- Agent角色要明确单一
- Task描述要清晰具体
- 合理使用任务依赖
- 善用工具增强能力
