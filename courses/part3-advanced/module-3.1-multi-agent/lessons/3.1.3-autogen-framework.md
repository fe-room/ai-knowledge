# 3.1.3 AutoGen框架实战

> 使用AutoGen构建多Agent对话系统

## 一、AutoGen概述

### 1.1 什么是AutoGen？

**AutoGen** 是微软开发的多Agent对话框架，支持Agent间的自动对话。

```
AutoGen核心特点：

1. 对话式交互
   Agent之间通过对话协作

2. 自动化程度高
   Agent可以自动推进对话

3. 人类参与
   支持人类介入对话

4. 代码执行
   Agent可以执行代码
```

### 1.2 安装

```python
pip install pyautogen
```

## 三、基础对话

### 3.1 两Agent对话

```python
"""
两个Agent对话
"""

import autogen

# 配置
config_list = [{
    "model": "gpt-4",
    "api_key": "your-api-key"
}]

# 创建助手
assistant = autogen.AssistantAgent(
    name="assistant",
    system_message="你是一个有用的AI助手。",
    llm_config={"config_list": config_list}
)

# 创建用户代理
user_proxy = autogen.UserProxyAgent(
    name="user",
    human_input_mode="ALWAYS",
    max_consecutive_auto_reply=10,
    code_execution_config=False
)

# 开始对话
user_proxy.initiate_chat(
    assistant,
    message="请解释什么是机器学习？"
)
```

## 四、Group Chat

### 4.1 创建群聊

```python
"""
AutoGen Group Chat
"""

import autogen

# 创建多个Agent
planner = autogen.AssistantAgent(
    name="Planner",
    system_message="你是一个任务规划专家，负责分解复杂任务。",
    llm_config={"config_list": config_list}
)

engineer = autogen.AssistantAgent(
    name="Engineer",
    system_message="你是一个工程师，负责执行具体任务。",
    llm_config={"config_list": config_list}
)

reviewer = autogen.AssistantAgent(
    name="Reviewer",
    system_message="你是一个审核员，负责检查和验证结果。",
    llm_config={"config_list": config_list}
)

# 用户代理
user_proxy = autogen.UserProxyAgent(
    name="User",
    human_input_mode="NEVER",
    code_execution_config={"work_dir": "workspace"}
)

# 创建群聊
groupchat = autogen.GroupChat(
    agents=[user_proxy, planner, engineer, reviewer],
    messages=[],
    max_round=20
)

# 群聊管理器
manager = autogen.GroupChatManager(
    groupchat=groupchat,
    llm_config={"config_list": config_list}
)

# 开始群聊
user_proxy.initiate_chat(
    manager,
    message="帮我开发一个简单的待办事项应用"
)
```

### 4.2 群聊控制

```python
"""
群聊流程控制
"""

class CustomGroupChat(autogen.GroupChat):
    """自定义群聊"""

    def select_speaker(self, last_speaker, messages):
        """选择下一个发言者"""
        # 自定义发言者选择逻辑
        if last_speaker.name == "Planner":
            return self.agent_by_name("Engineer")
        elif last_speaker.name == "Engineer":
            return self.agent_by_name("Reviewer")
        elif last_speaker.name == "Reviewer":
            return self.agent_by_name("User")
        else:
            return self.agent_by_name("Planner")

# 使用自定义群聊
custom_groupchat = CustomGroupChat(
    agents=[user_proxy, planner, engineer, reviewer],
    messages=[],
    max_round=20
)
```

## 五、高级特性

### 5.1 工具使用

```python
"""
AutoGen工具
"""

from autogen import register_function

# 定义工具
def search_web(query: str) -> str:
    """搜索网络"""
    # 实现搜索逻辑
    return f"搜索结果: {query}"

def calculate(expression: str) -> float:
    """计算"""
    return eval(expression)

# 注册工具
register_function(
    search_web,
    caller=assistant,
    executor=user_proxy,
    description="搜索网络获取信息"
)

register_function(
    calculate,
    caller=assistant,
    executor=user_proxy,
    description="执行数学计算"
)
```

## 总结

### 核心要点

1. **Agent类型**：AssistantAgent、UserProxyAgent
2. **对话模式**：两Agent、Group Chat
3. **代码执行**：自动执行Python代码
4. **工具集成**：注册自定义工具
5. **状态管理**：保持对话上下文

### 与CrewAI对比

| 特性 | AutoGen | CrewAI |
|------|---------|--------|
| 协作方式 | 对话式 | 任务驱动 |
| 人类参与 | 强支持 | 可选 |
| 代码执行 | 内置 | 需工具 |
| 学习曲线 | 中等 | 较低 |
