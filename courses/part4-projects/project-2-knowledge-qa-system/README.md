- 多格式文档处理

  - PyPDF2（PDF）
  - python-docx（Word）
  - beautifulsoup4（HTML）
  - langchain（统一接口）

向量存储:
  - OpenAI Embeddings
  - Qdrant（向量数据库）

检索框架:
  - LlamaIndex
  - LangChain

  - OpenAI GPT-4
  - Claude

## 评估指标

### 检索质量

| 指标 | 目标值 | 说明 |
|------|--------|------|
| Recall@5 | ≥0.85 | 前5个结果召回率 |
| Precision@5 | ≥0.80 | 前5个结果准确率 |
| MRR | ≥0.75 | 平均倒数排名 |
| NDCG@10 | ≥0.80 | 归一化折损累计增益 |

### 生成质量

| 指标 | 目标值 | 说明 |
|------|--------|------|
| 准确性 | ≥0.90 | 答案正确性 |
| 完整性 | ≥0.85 | 信息完整度 |
| 相关性 | ≥0.90 | 答案相关性 |
| 流畅性 | ≥4.0/5 | 语言流畅度 |

### 性能指标

| 指标 | 目标值 | 说明 |
|------|--------|------|
| 索引速度 | ≥100 docs/min | 文档索引速度 |
| 检索延迟 | <500ms | 检索响应时间 |
| 问答延迟 | <3s | 完整问答时间 |
