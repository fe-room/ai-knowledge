## 核心架构

```
多Agent协作系统架构：

┌──────────────────────────────────────────┐
│              Orchestrator                 │
│         任务分发 + 状态管理               │
└─────────────────┬────────────────────────┘
                  │
    ┌─────────────┼─────────────┐
    │             │             │
┌───┴───┐    ┌───┴───┐    ┌───┴───┐
│Planner│    │Researcher   │Writer │
│       │────│             │       │
└───────┘    └─────┬───────┘───────┘
                   │
              ┌────┴────┐
              │Reviewer │
              │         │
              └─────────┘
```

---

## Agent角色

### Planner
- 分析任务需求
- 制定执行计划
- 分配子任务
- 跟踪进度

### Researcher
- 收集信息资料
- 检索相关知识
- 整理数据
- 提供素材

### Writer
- 生成内容
- 组织结构
- 格式处理
- 润色优化

### Reviewer
- 审核质量
- 检查准确性
- 提供反馈
- 建议修改

## 工作流程

```
任务输入
    │
    ▼
┌─────────┐
│ Planner │ 分析任务，制定计划
└────┬────┘
     │
     ▼
┌─────────────┐
│ Researcher  │ 收集资料
└──────┬──────┘
       │
       ▼
┌─────────┐
│ Writer  │ 生成内容
└────┬────┘
     │
     ▼
┌──────────┐
│ Reviewer │ 审核修改
└────┬─────┘
     │
     ▼
  任务输出
```
